# AboutMe - Personal Portfolio

This is my personal portfolio website built using HTML, CSS, and JavaScript with a JSON-driven data structure.

## 🚀 Overview
The website dynamically renders content from a structured JSON file, allowing easy updates without modifying the UI code.

## 🧠 Key Features
- JSON-based data source for profile content
- Responsive static frontend (HTML/CSS/JS)
- Modular structure for easy maintenance
- Clean separation between data and presentation

## 📁 Structure
- `index.html` - Main UI
- `styles.css` - Styling
- `main.js` - Logic to render JSON data
- `/data/profile.json` - Profile and content data source

## 💼 Purpose
This project serves as:
- My online CV / portfolio
- A showcase of frontend structure and data-driven design
- A continuously evolving personal profile
